# **App Name**: OrçaDiária

## Core Features:

- Budget Creation: Create new budgets by entering client details, task descriptions, and work periods.
- Configurable Daily Rates: Allow users to set up daily rates by configuring the start and end times, total workload, and value.
- Automated Budget Calculation: Automatically calculate the total budget based on configured daily rates and work periods.
- Dashboard Overview: Provide a dashboard displaying created budgets with statuses (e.g., active, completed, canceled).
- Financial Summary: Show a financial summary including total created budget values, received amounts, and outstanding balances.
- AI-Powered Task Suggestion Tool: Suggest tasks based on client information using generative AI; decides whether suggestions will be helpful based on available information.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey professionalism and reliability.
- Background color: Light gray (#ECEFF1) to create a clean and neutral backdrop.
- Accent color: Green (#81C784) for positive actions and completed tasks.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use minimalist and intuitive icons for navigation and status indicators.
- Implement a responsive layout to ensure functionality across web and Android devices.
- Incorporate subtle transitions and animations to enhance user experience without being distracting.
